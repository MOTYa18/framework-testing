Был разработан собственный UI-фреймворк на Python с использованием Selenium WebDriver и pytest.

UI-фреймворк для автоматизированного тестирования веб-приложения.
Основой фреймворка стала архитектура Page Object Model.

В проекте были реализованы:
отдельные классы для каждой страницы;
базовый класс с общими методами;
отдельный файл с локаторами;
фикстуры pytest для запуска браузера;
позитивные и негативные тесты;
ожидания элементов через WebDriverWait.

Преимущества реализованного фреймворка:
уменьшение повторяющегося кода;
более удобная поддержка тестов;
возможность быстро добавлять новые тестовые сценарии;
улучшение читаемости проекта;
возможность повторного использования методов.


Были установлены следующие библиотеки:
selenium;
pytest;
